# PrePAU Lengua Castellana · 2º Bachillerato

Aplicación interactiva de práctica para la PAU de Lengua Castellana y Literatura, desarrollada para Proa Escola (curso 24/25).

## Apartados

- **Expresión escrita** — Registro, argumentación, corrección, estilo indirecto, puntuación, conectores y transformaciones
- **Pares mínimos** — Diferencias gramaticales entre pares de oraciones
- **Análisis inverso** — Construye oraciones con los elementos indicados
- **Funciones sintácticas** — Identifica funciones, analiza pronombres y relaciona sintagmas
- **Oraciones compuestas** — Modalidad, subordinadas, comparación de estructuras
- **Léxico** — Vocabulario, sinónimos, parónimos
- **Morfología** — Jerarquías de formación, análisis morfológico

## Características

- 147 ejercicios tipo test, respuesta abierta y completar
- Corrección automática con IA (Anthropic Claude)
- Vista «uno a uno» y vista «lista»
- **Progreso guardado automáticamente** en el navegador (localStorage)
- Totalmente responsive (móvil y escritorio)

## Despliegue en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En *Source*, selecciona **Branch: main**, carpeta **/ (root)**
4. Guarda — en unos minutos estará disponible en `https://tuusuario.github.io/nombre-repo`

## Desarrollo local

Abre `index.html` directamente en el navegador. No se necesita servidor ni instalación.
